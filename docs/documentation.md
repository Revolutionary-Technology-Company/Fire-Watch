How Engineers Use the Documentation Engine
Once added to your workspace, plant and commissioning engineers can dynamically query manuals right inside their terminal shell using subcommands.
1. Display General Engine Help Options:

python doc_engine.py --help


2. Pull up Server and Core Dependency Rules:

python doc_engine.py server


3. View JSON Configuration Objects and Camera Tuning Thresholds:

python doc_engine.py camera


4. Fetch the Edwards FireWorks Port & Delimiter Field Mappings:

python doc_engine.py edwards


5. Output the Windows Service Control Administration Cheat Sheet:

python doc_engine.py commands
